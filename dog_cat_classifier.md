# 🐶🐱 Dog-Cat Classifier

A deep learning project built with the **Fastai** library using the **Oxford-IIIT Pets Dataset** to classify images of dogs and cats.  
This model achieves high accuracy and demonstrates the power of transfer learning for image classification tasks.

---

## 📌 Overview
The project leverages Fastai’s vision module to fine-tune a convolutional neural network (CNN) on the Oxford-IIIT Pets dataset.  
The dataset contains images of cats and dogs across multiple breeds, making it a robust benchmark for binary classification tasks.

---

## ⚙️ Features
- Image classification: predicts whether an image contains a dog or a cat  
- High performance: trained with transfer learning for strong accuracy  
- Easy to use: simple pipeline for training and inference  
- Extensible: can be adapted to multi-class classification (different breeds)  

---

## 🚀 Getting Started
1. Clone the repository  
2. Install dependencies listed in `requirements.txt`  
3. Download the Oxford-IIIT Pets dataset  
4. Train the model or load the pre-trained weights  
5. Run predictions on new images  

---

## 📊 Results
- Achieved strong accuracy on validation data  
- Robust performance across diverse breeds  
- Example predictions:  
  - Image of a Labrador → Dog  
  - Image of a Persian → Cat
