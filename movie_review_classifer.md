# 🎬 Movie Review Classifier

A deep learning project built with the **Fastai** library to classify movie reviews into **positive** or **negative** sentiments.  
This model achieves an impressive **92% accuracy**, making it a reliable tool for sentiment analysis in the film domain.

---

## 📌 Overview
This project fine-tunes a language model using Fastai’s text classification framework.  
By training on a dataset of labeled movie reviews, the classifier learns sentiment patterns and generalizes well to unseen reviews.

---

## ⚙️ Features
- Sentiment analysis: predicts whether a review is positive or negative  
- High accuracy: 92% validation accuracy  
- Transfer learning: built on pre-trained language models  
- Extensible: adaptable to other domains like product reviews or social media comments  

---

## 🚀 Getting Started
1. Clone the repository  
2. Install dependencies listed in `requirements.txt`  
3. Load the trained model file (`movie_review_classifier.pkl`)  
4. Run predictions on new reviews  

---

## 📊 Results
- Validation accuracy: 92%  
- Confusion matrix: strong performance across both classes  
- Example predictions:  
  - "A masterpiece of storytelling." → Positive  
  - "The plot was dull and predictable." → Negative
